# DESTool

[English](#English) | [中文](#中文)

# 中文

DESTool 是一个纯 JavaScript 实现的 DES (ECB 电子密码本模式) 的在线加解密工具。UI 使用 MDUI 构建。

## 功能特性

+ 纯文本加密、解密
+ 文件加密、解密
+ 快捷拷贝至剪贴板
+ 加密结果以 Hex 输出，可以指定分隔符
+ 支持非标准长度的密码

# English

DESTool is a JavaScript-based DES tool works on ECB (Electronic code book) mode. Web UI is constructed by MDUI.

## Features

+ Text encryption and decryption.
+ File encryption and decryption.
+ Copy to paste button.
+ Print encryption result as Hex with specified separator.
+ Support for non-standard key length.
